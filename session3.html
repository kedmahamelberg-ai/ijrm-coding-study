Codebook Reference

What you are coding

Each post mentions AI (ChatGPT, GenAI, etc.). For each post, assign 2 codes:

  • Consumer Experience: what is the human in the post experiencing?
  • AI Role: what is the AI doing?

The two codes are independent — answer each one based on what the post says.

Consumer Experience — five options

CodeMeaning
EXTENSIONAI helps person do something they could already do — faster/easier/at scale. Could they do this without AI? YES → EXTENSION.
EXPANSIONAI gives a NEW ability. Could they do this without AI? NO → EXPANSION.
RESTRICTIONAI limits autonomy or forces behavior changes. "I have to change what I do because of AI."
REDUCTIONAI erodes existing skills. "AI is making us worse at X."
NEUTRALNo clear consumer experience (news, marketing, announcement). Default when unsure.

Examples

  • "I use ChatGPT to write my emails 10x faster" → EXTENSION
  • "ChatGPT taught me a new language in a weekend" → EXPANSION
  • "BEWARE: ChatGPT can find your address from photos. Stop posting outdoor pics." → RESTRICTION
  • "Students using ChatGPT show less critical thinking" → REDUCTION
  • "AI Summit 2025 next week!" → NEUTRAL

AI Role — five options

CodeMeaning
AI_EXTENSIONAI works well, produces useful outputs as intended.
AI_EXPANSIONAI/its company is growing, profiting, expanding reach.
AI_RESTRICTIONAI blocks, refuses, censors, or resists.
AI_REDUCTIONAI fails, breaks, hallucinates, or degrades.
NEUTRALNo clear AI role expressed.

The hardest call: EXTENSION vs EXPANSION

Could the person plausibly have done this task before AI existed (just slower or with more effort)?

  • YES → EXTENSION (existing ability amplified)
  • NO → EXPANSION (new ability created)

Speed/scale/productivity emphasis → EXTENSION. Learning/new skills → EXPANSION.

Important rules

  1. Code what the post SAYS, not whether it's true.
  2. Code the dominant framing. If genuinely balanced → NEUTRAL.
  3. NEUTRAL is the safe default. If you're stuck or unclear, NEUTRAL is correct.
  4. Sarcasm/jokes: code the underlying point.
  5. Spam, marketing, off-topic, non-English: NEUTRAL on both AND tick "Off-topic: Yes".

Confidence ratings

  • HIGH — signal is obvious, anyone reading would code the same way
  • MEDIUM — code is right, some interpretation involved
  • LOW — guessing your best (consider whether NEUTRAL is more appropriate)
